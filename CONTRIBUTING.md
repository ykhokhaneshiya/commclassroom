### 2.) Clone The Repo
Clone repo to your machine

```bash

git clone https://github.com/<YOUR_USERNAME>/commclassroom.git

```

Navigate 

```bash

cd commclassroom

```

Install Dependencies 

```bash

npm install

```

### 3.) Setup Remote

```bash

git remote add upstream https://github.com/commclassroom/commclassroom.git

```
To Verify 

```bash

git remote -v

```
